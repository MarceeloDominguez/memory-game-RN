## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/memory-game-RN.git
```

Go to the project directory

```bash
  cd memory-game-RN
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
